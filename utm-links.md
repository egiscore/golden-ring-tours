## Цели для отслеживания в Яндекс.Метрике

### Основные цели:
1. **click_get_discount** - Клик по кнопке "Получить скидку 5%"
2. **click_book_tour** - Клик по кнопке "Забронировать со скидкой"
3. **form_submit** - Отправка формы бронирования
4. **page_view** - Просмотр страницы (автоматически)

### Настройка целей в Метрике:
```javascript
// Цель 1: Клик на получение скидки
ID: click_get_discount
Условие: JavaScript событие
Идентификатор: click_get_discount

// Цель 2: Клик на бронирование тура
ID: click_book_tour
Условие: JavaScript событие
Идентификатор: click_book_tour

// Цель 3: Отправка формы
ID: form_submit
Условие: JavaScript событие
Идентификатор: form_submit
```

## Сегменты для анализа

### По городам:
- Москва: `utm_campaign содержит "_moscow"`
- СПб: `utm_campaign содержит "_spb"`
- Казань: `utm_campaign содержит "_kazan"`
- И так далее для каждого города

### По каналам:
- Поиск: `utm_medium = cpc`
- Ретаргетинг: `utm_medium = display`
- Соцсети: `utm_medium = social`

### По эффективности:
- Конверсия в клик: `Достигли цели click_get_discount`
- Конверсия в бронь: `Достигли цели click_book_tour`
- Конверсия в заявку: `Достигли цели form_submit`
